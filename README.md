# Job Listing Application - Backend API

A robust, backend-only RESTful API built to power a job board platform. This application allows users to post new job requirements and features advanced searching capabilities, including full-text search, backed by a cloud-hosted MongoDB database.

## 🚀 Features

* **Job Posting:** Submit and store new job listings with detailed requirements.
* **Job Retrieval:** Fetch available job listings from the database.
* **Full-Text Search:** Powerful text-based search functionality to quickly find relevant jobs based on keywords, descriptions, or titles.
* **Interactive API Documentation:** Fully integrated Swagger UI for easy testing and exploration of the API endpoints without needing a frontend.

## 🛠️ Technologies Used

* **Framework:** Spring Boot (Java)
* **Database:** MongoDB Atlas (Cloud)
* **API Documentation:** Swagger / OpenAPI
* **Build Tool:** Maven

## 📋 Prerequisites

Before you begin, ensure you have the following installed on your local machine:
* JDK 17 or higher
* Maven
* A MongoDB Atlas account and cluster
